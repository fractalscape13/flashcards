  • JavaScript (JS)

    An object-oriented computer programming language commonly used to create interactive effects within web browsers.

  • Operators

    Mathematical signs like +, -, /, *, and %.

  • Variables

    Containers to show info.

  • Variable naming conventions

    In computer programming, a naming convention is a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

  • Functions

    Block of code that performs an action and returns a result.

  • Methods

    The action run on a piece of data.

  • Arguments

    Values passed to the function when it is invoked.

  • Parameters

    Parameters are variables listed as a part of the function definition.

  • Return

    The return statement stops the execution of a function and returns a value from that function.

  • Chaining methods

    Otherwise known as "cascading," refers to repeatedly calling one method after another on an object in one continuous line of code.

  • Strings

    The text within quotation marks.

  • Booleans

    What is true or false.

  • Undefined

    A primitive value used when a variable has not been assigned a value.

  • NaN

    Stands for "Not a Number". Actually is regarded as a number type within JS.

  • Escape

    This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

  • Alert

    Displays an alert box with a specified message and an OK button. Used to make sure that information comes through to the user.

  • jQuery

    A concise and fast JavaScript library that can be used to simplify ebent handling and speedy website development.

  • Attributes

    These define how a JavaScript property can be accessed.
